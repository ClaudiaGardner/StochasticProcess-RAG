# 习题2.4

## 题目

假设光临某商店的顾客数服从参数为4 人每小时的泊松过程。已知商店早上9:00 开业，求： (1) 到9:30 为止恰好一位顾客光临的概率？ (2) 到11:30 为止总共到达5 位顾客的概率？

## 解答

## 解题过程

### 题目分析

本题考查泊松过程的基本性质。顾客到达服从参数为 $\lambda = 4$ 人/小时的泊松过程，需要计算不同时间段内到达特定顾客数的概率。泊松过程的关键性质是：在长度为 $t$ 的时间区间内，到达的顾客数 $N(t)$ 服从参数为 $\lambda t$ 的泊松分布。

### 解题过程

**泊松过程的基本性质**

对于参数为 $\lambda$ 的泊松过程，在时间区间 $[0, t]$ 内到达 $k$ 个顾客的概率为：

$$P(N(t) = k) = \frac{(\lambda t)^k}{k!} e^{-\lambda t}, \quad k = 0, 1, 2, \ldots$$

本题中 $\lambda = 4$ 人/小时。

---

**(1) 到9:30为止恰好一位顾客光临的概率**

从9:00到9:30，时间长度为 $t = 0.5$ 小时。

需要求 $P(N(0.5) = 1)$。

计算参数：
$$\lambda t = 4 \times 0.5 = 2$$

代入泊松分布公式，$k = 1$：

$$P(N(0.5) = 1) = \frac{2^1}{1!} e^{-2} = 2e^{-2}$$

计算数值：
$$P(N(0.5) = 1) = 2e^{-2} = \frac{2}{e^2} \approx 2 \times 0.1353 = 0.2707$$

---

**(2) 到11:30为止总共到达5位顾客的概率**

从9:00到11:30，时间长度为 $t = 2.5$ 小时。

需要求 $P(N(2.5) = 5)$。

计算参数：
$$\lambda t = 4 \times 2.5 = 10$$

代入泊松分布公式，$k = 5$：

$$P(N(2.5) = 5) = \frac{10^5}{5!} e^{-10}$$

计算阶乘和幂次：
$$5! = 120, \quad 10^5 = 100000$$

因此：
$$P(N(2.5) = 5) = \frac{100000}{120} e^{-10} = \frac{2500}{3} e^{-10}$$

计算数值：
$$P(N(2.5) = 5) = \frac{2500}{3} \times e^{-10} \approx 833.33 \times 0.0000454 \approx 0.0378$$

### 答案

**(1)** 到9:30为止恰好一位顾客光临的概率为：
$$P(N(0.5) = 1) = 2e^{-2} \approx 0.2707$$

**(2)** 到11:30为止总共到达5位顾客的概率为：
$$P(N(2.5) = 5) = \frac{2500}{3} e^{-10} = \frac{10^5}{120} e^{-10} \approx 0.0378$$

### 知识延伸

**泊松过程的重要性质：**

1. **增量的独立性**：不相交时间区间内的到达数相互独立
2. **增量的平稳性**：到达数的分布只依赖于时间区间的长度，不依赖于起始时刻
3. **泊松分布**：时间长度为 $t$ 的区间内到达数 $N(t) \sim \text{Poisson}(\lambda t)$

**重要公式：**

- 泊松分布概率质量函数：$P(N(t) = k) = \frac{(\lambda t)^k}{k!} e^{-\lambda t}$
- 期望：$E[N(t)] = \lambda t$
- 方差：$\text{Var}(N(t)) = \lambda t$

**解题技巧：**

1. 准确计算时间区间长度，注意单位统一
2. 计算 $\lambda t$ 作为泊松分布的参数
3. 对于数值计算，可保留精确表达式或计算近似值

**易错点：**

- 时间单位不统一（如 $\lambda$ 是人/小时，时间要用小时表示）
- 混淆泊松过程的参数 $\lambda$ 与泊松分布的参数 $\lambda t$
