# 习题2.6

## 题目

令N = (N(t)，t ≥0) 是参数为2 的泊松过程，求： (1) P(N(1) ≤2) (2) P(N(1) = 1，N(2) = 3) (3) P(N(1) ≥2|N(1) ≥1)

## 解答

### 题目分析

本题考查泊松过程的基本性质。已知 $N(t)$ 是参数（强度）为 $\lambda=2$ 的泊松过程，需要计算三个概率：(1) 时刻1时事件数不超过2的概率；(2) 时刻1有1个事件且时刻2有3个事件的联合概率；(3) 在时刻1至少有1个事件的条件下至少有2个事件的条件概率。核心是利用泊松分布和泊松过程的独立增量性质。

### 解题过程

**基础知识**：对于强度为 $\lambda$ 的泊松过程，$N(t) \sim \text{Poisson}(\lambda t)$，即
$$P(N(t) = k) = \frac{(\lambda t)^k e^{-\lambda t}}{k!}, \quad k = 0,1,2,\ldots$$

本题中 $\lambda = 2$。

**(1) 求 $P(N(1) \leq 2)$**

$N(1) \sim \text{Poisson}(2 \times 1) = \text{Poisson}(2)$

$$P(N(1) \leq 2) = P(N(1) = 0) + P(N(1) = 1) + P(N(1) = 2)$$

计算各项：
$$P(N(1) = 0) = \frac{2^0 e^{-2}}{0!} = e^{-2}$$

$$P(N(1) = 1) = \frac{2^1 e^{-2}}{1!} = 2e^{-2}$$

$$P(N(1) = 2) = \frac{2^2 e^{-2}}{2!} = \frac{4e^{-2}}{2} = 2e^{-2}$$

因此：
$$P(N(1) \leq 2) = e^{-2} + 2e^{-2} + 2e^{-2} = 5e^{-2}$$

**(2) 求 $P(N(1) = 1, N(2) = 3)$**

利用泊松过程的独立增量性质。事件 $\{N(1) = 1, N(2) = 3\}$ 等价于 $\{N(1) = 1, N(2) - N(1) = 2\}$。

由于 $N(1)$ 与增量 $N(2) - N(1)$ 独立，且：
- $N(1) \sim \text{Poisson}(2)$
- $N(2) - N(1) \sim \text{Poisson}(2 \times (2-1)) = \text{Poisson}(2)$

所以：
$$P(N(1) = 1, N(2) = 3) = P(N(1) = 1) \cdot P(N(2) - N(1) = 2)$$

$$= \frac{2^1 e^{-2}}{1!} \cdot \frac{2^2 e^{-2}}{2!}$$

$$= 2e^{-2} \cdot 2e^{-2} = 4e^{-4}$$

**(3) 求 $P(N(1) \geq 2 | N(1) \geq 1)$**

使用条件概率公式：
$$P(N(1) \geq 2 | N(1) \geq 1) = \frac{P(N(1) \geq 2, N(1) \geq 1)}{P(N(1) \geq 1)}$$

由于 $\{N(1) \geq 2\} \subset \{N(1) \geq 1\}$，所以：
$$P(N(1) \geq 2 | N(1) \geq 1) = \frac{P(N(1) \geq 2)}{P(N(1) \geq 1)}$$

计算分子：
$$P(N(1) \geq 2) = 1 - P(N(1) \leq 1) = 1 - [P(N(1) = 0) + P(N(1) = 1)]$$
$$= 1 - [e^{-2} + 2e^{-2}] = 1 - 3e^{-2}$$

计算分母：
$$P(N(1) \geq 1) = 1 - P(N(1) = 0) = 1 - e^{-2}$$

因此：
$$P(N(1) \geq 2 | N(1) \geq 1) = \frac{1 - 3e^{-2}}{1 - e^{-2}}$$

### 答案

(1) $P(N(1) \leq 2) = 5e^{-2}$

(2) $P(N(1) = 1, N(2) = 3) = 4e^{-4}$

(3) $P(N(1) \geq 2 | N(1) \geq 1) = \dfrac{1 - 3e^{-2}}{1 - e^{-2}}$

### 知识延伸

**泊松过程的关键性质**：
1. **计数分布**：$N(t) \sim \text{Poisson}(\lambda t)$，即 $P(N(t) = k) = \frac{(\lambda t)^k e^{-\lambda t}}{k!}$
2. **独立增量性**：不相交时间区间上的增量相互独立
3. **平稳增量性**：增量 $N(t+s) - N(s) \sim \text{Poisson}(\lambda t)$，只依赖于时间长度

**解题技巧**：
- 计算联合概率时，利用独立增量性将其转化为增量的概率乘积
- 条件概率问题注意集合的包含关系
- 计算 $P(N(t) \geq k)$ 时，通常用补集 $1 - P(N(t) \leq k-1)$ 更简便

**易错点**：
- 混淆 $N(2) = 3$ 与增量 $N(2) - N(1)$ 的关系
- 条件概率中忘记检查事件的包含关系
