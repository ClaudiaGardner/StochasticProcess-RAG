# 习题2.5

## 题目

令N = (N(t)，t ≥0) 是参数为2 的泊松过程，求： (1) P(N(1) = 2) (2) P(N(1) = 2，N(3) = 6) (3) P(N(1) = 2|N(3) = 6)

## 解答

### 题目分析

本题考查泊松过程的基本性质。已知 $N(t)$ 是参数（强度）为 $\lambda = 2$ 的泊松过程，需要求解三个概率问题：单个时刻的概率、联合概率和条件概率。核心是利用泊松过程的定义、独立增量性和条件分布性质。

### 解题过程

**泊松过程的基本性质**：
对于参数为 $\lambda$ 的泊松过程 $N(t)$，有 $N(t) \sim \text{Poisson}(\lambda t)$，即
$$P(N(t) = k) = \frac{(\lambda t)^k e^{-\lambda t}}{k!}$$

本题中 $\lambda = 2$。

**(1) 求 $P(N(1) = 2)$**

直接应用泊松分布公式，$t=1$，$k=2$：
$$P(N(1) = 2) = \frac{(2 \cdot 1)^2 e^{-2 \cdot 1}}{2!} = \frac{4e^{-2}}{2} = 2e^{-2}$$

**(2) 求 $P(N(1) = 2, N(3) = 6)$**

利用泊松过程的独立增量性。注意到：
$$P(N(1) = 2, N(3) = 6) = P(N(1) = 2, N(3) - N(1) = 4)$$

由于 $N(1)$ 和 $N(3) - N(1)$ 是独立的增量，且：
- $N(1) \sim \text{Poisson}(2 \cdot 1) = \text{Poisson}(2)$
- $N(3) - N(1) \sim \text{Poisson}(2 \cdot 2) = \text{Poisson}(4)$

因此：
$$P(N(1) = 2, N(3) = 6) = P(N(1) = 2) \cdot P(N(3) - N(1) = 4)$$

$$= \frac{2^2 e^{-2}}{2!} \cdot \frac{4^4 e^{-4}}{4!}$$

$$= \frac{4e^{-2}}{2} \cdot \frac{256e^{-4}}{24}$$

$$= 2e^{-2} \cdot \frac{32e^{-4}}{3}$$

$$= \frac{64e^{-6}}{3}$$

**(3) 求 $P(N(1) = 2 | N(3) = 6)$**

使用条件概率公式：
$$P(N(1) = 2 | N(3) = 6) = \frac{P(N(1) = 2, N(3) = 6)}{P(N(3) = 6)}$$

先计算 $P(N(3) = 6)$：
$$P(N(3) = 6) = \frac{(2 \cdot 3)^6 e^{-2 \cdot 3}}{6!} = \frac{6^6 e^{-6}}{720} = \frac{46656e^{-6}}{720} = \frac{1944e^{-6}}{30} = \frac{324e^{-6}}{5}$$

因此：
$$P(N(1) = 2 | N(3) = 6) = \frac{\frac{64e^{-6}}{3}}{\frac{324e^{-6}}{5}} = \frac{64}{3} \cdot \frac{5}{324} = \frac{320}{972} = \frac{80}{243}$$

**另一种方法（利用泊松过程的条件分布性质）**：

已知 $N(3) = 6$，则 $N(1)$ 服从二项分布 $B(6, \frac{1}{3})$，因为：
$$P(N(1) = k | N(3) = n) = \binom{n}{k} \left(\frac{1}{3}\right)^k \left(\frac{2}{3}\right)^{n-k}$$

所以：
$$P(N(1) = 2 | N(3) = 6) = \binom{6}{2} \left(\frac{1}{3}\right)^2 \left(\frac{2}{3}\right)^4 = 15 \cdot \frac{1}{9} \cdot \frac{16}{81} = \frac{240}{729} = \frac{80}{243}$$

### 答案

(1) $P(N(1) = 2) = 2e^{-2}$

(2) $P(N(1) = 2, N(3) = 6) = \frac{64e^{-6}}{3}$

(3) $P(N(1) = 2 | N(3) = 6) = \frac{80}{243}$

### 知识延伸

**泊松过程的关键性质**：
1. **泊松分布**：$N(t) \sim \text{Poisson}(\lambda t)$
2. **独立增量性**：不相交时间区间上的增量相互独立
3. **平稳增量性**：增量的分布只依赖于时间长度
4. **条件分布**：给定 $N(t) = n$，$0 < s < t$ 时刻前的事件数 $N(s) | N(t) = n \sim B(n, \frac{s}{t})$

**解题技巧**：
- 联合概率问题：利用独立增量性分解
- 条件概率问题：可用定义计算，也可直接用条件分布的二项性质

**易错点**：
- 混淆 $N(t)$ 和增量 $N(t) - N(s)$ 的分布参数
- 忘记泊松过程的条件分布是二项分布而非泊松分布
