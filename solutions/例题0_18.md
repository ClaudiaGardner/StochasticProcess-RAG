# 例题0.18

## 题目

投掷两个均匀的骰子。 随机变量Xi，i ∈ {1，2}表示第i 个骰子的点数，Xi ∈ {1，2，3，4，5， 6}。令X = X1 + X2，求： （1）P(X = 2 |X1 = 6) ； （2）P(X1 = 2 |X2 = 2) ； （3）P(X1 = 1 |X = 2) 。

### 题目分析
本题考查概率论中的基础概念，主要涉及**样本空间**的构建、**随机变量**的定义、**事件的独立性**以及**条件概率**的计算。题目设定为投掷两个均匀骰子，这是一个典型的离散型概率模型。核心在于理解条件概率的定义 $P(A|B) = \frac{P(AB)}{P(B)}$，以及如何根据已知条件缩小样本空间来求解概率。

### 解题过程

首先定义样本空间 $\Omega$。投掷两个均匀骰子，每个骰子的结果 $X_i \in \{1, 2, 3, 4, 5, 6\}$。
样本空间由所有可能的有序对 $(x_1, x_2)$ 组成，总数为：
$$|\Omega| = 6 \times 6 = 36$$
由于骰子是均匀的，每个基本事件发生的概率为 $\frac{1}{36}$。
随机变量 $X_1$ 和 $X_2$ 相互独立。

#### （1）求解 $P(X = 2 |X_1 = 6)$

根据条件概率公式：
$$P(X = 2 |X_1 = 6) = \frac{P(X = 2 \cap X_1 = 6)}{P(X_1 = 6)}$$

**分母分析**：
$X_1 = 6$ 的事件包含 6 个基本事件：$\{(6,1), (6,2), (6,3), (6,4), (6,5), (6,6)\}$。
$$P(X_1 = 6) = \frac{6}{36} = \frac{1}{6}$$

**分子分析**：
事件 $\{X = 2 \cap X_1 = 6\}$ 意味着 $X_1 + X_2 = 2$ 且 $X_1 = 6$。
代入 $X_1 = 6$，得到 $6 + X_2 = 2$，即 $X_2 = -4$。
然而 $X_2 \in \{1, 2, 3, 4, 5, 6\}$，不可能取值为 -4。
因此，该事件为空集，概率为 0。
$$P(X = 2 \cap X_1 = 6) = 0$$

**计算结果**：
$$P(X = 2 |X_1 = 6) = \frac{0}{1/6} = 0$$

---

#### （2）求解 $P(X_1 = 2 |X_2 = 2)$

**方法一：利用独立性**
因为投掷两个骰子互不干扰，随机变量 $X_1$ 与 $X_2$ 相互独立。根据独立性的性质，事件 $X_1=2$ 的发生与否与 $X_2=2$ 无关。
$$P(X_1 = 2 |X_2 = 2) = P(X_1 = 2)$$
对于均匀骰子：
$$P(X_1 = 2) = \frac{1}{6}$$

**方法二：利用定义计算**
$$P(X_1 = 2 |X_2 = 2) = \frac{P(X_1 = 2 \cap X_2 = 2)}{P(X_2 = 2)}$$
分子为事件 $\{(2, 2)\}$，概率为 $\frac{1}{36}$。
分母为事件 $X_2=2$，包含 6 个样本点，概率为 $\frac{6}{36} = \frac{1}{6}$。
$$P(X_1 = 2 |X_2 = 2) = \frac{1/36}{1/6} = \frac{1}{6}$$

---

#### （3）求解 $P(X_1 = 1 |X = 2)$

根据条件概率公式：
$$P(X_1 = 1 |X = 2) = \frac{P(X_1 = 1 \cap X = 2)}{P(X = 2)}$$

**分母分析**：
事件 $X = 2$ 即 $X_1 + X_2 = 2$。
在样本空间中，满足两数之和为 2 的样本点只有 $\{(1, 1)\}$。
$$P(X = 2) = \frac{1}{36}$$

**分子分析**：
事件 $\{X_1 = 1 \cap X = 2\}$ 意味着 $X_1 = 1$ 且 $X_1 + X_2 = 2$。
代入得 $1 + X_2 = 2 \Rightarrow X_2 = 1$。
对应的样本点为 $\{(1, 1)\}$。
$$P(X_1 = 1 \cap X = 2) = \frac{1}{36}$$

**计算结果**：
$$P(X_1 = 1 |X = 2) = \frac{1/36}{1/36} = 1$$

### 答案

（1）$P(X = 2 |X_1 = 6) = 0$
（2）$P(X_1 = 2 |X_2 = 2) = \frac{1}{6}$
（3）$P(X_1 = 1 |X = 2) = 1$

### 知识延伸

1.  **条件概率定义**：
    $$P(A|B) = \frac{P(AB)}{P(B)}, \quad P(B) > 0$$
    这是解决此类问题的通用公式。直观理解是：已知事件 $B$ 发生，样本空间缩减为 $B$，此时求 $A$ 在 $B$ 中所占的比例。

2.  **独立性 (Independence)**：
    若 $A, B$ 相互独立，则 $P(A|B) = P(A)$。在第（2）问中，识别出独立性可以直接得出答案，无需繁琐计算。

3.  **离散型随机变量的和**：
    对于 $X = X_1 + X_2$，确定 $P(X=k)$ 通常需要寻找满足 $i+j=k$ 的所有 $(i,j)$ 对。常见的易错点是漏掉样本点或错误估计样本空间大小（例如误以为 $X$ 的分布是均匀的，实际上两个骰子之和呈三角形分布，7的概率最大）。

4.  **边界条件检查**：
    在第（1）问中，通过简单的代数推导（$6+X_2=2$）即可发现超出定义域，直接判定概率为0，这是处理离散变量时的重要技巧。
