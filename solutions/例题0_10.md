# 例题0.10

## 题目

考虑一个概率空间。记 A，B，C 是三个事件。 （1）举例说明事件 A 与（B 与C 的和事件）独立，但 A 与 B 不独立，与 C 也不独立。 （2）设 A，B 两个事件的概率均大于 3/4。证明 A 与B 的积事件的概率大于 1/2。

### 题目分析

本题主要考查概率论基础中的**事件独立性**定义以及**概率加法公式**（或容斥原理）的应用。
1.  题目中涉及的符号 $A, B, C$ 为同一概率空间中的事件。
2.  第一问要求构造反例，旨在考察对独立性定义 $P(XY) = P(X)P(Y)$ 的深刻理解，说明事件与“和事件”的独立性并不蕴含与“子事件”的独立性。
3.  第二问考察概率的基本不等式性质，即如何通过边缘概率推导积事件（交事件）概率的下界，这与邦费罗尼不等式（Bonferroni inequalities）有关。

### 解题过程

#### （1）举例说明事件 A 与（B 与 C 的和事件）独立，但 A 与 B 不独立，与 C 也不独立。

我们需要构造一个概率空间 $\Omega$ 及三个事件 $A, B, C$，满足以下条件：
1.  $P(A \cap (B \cup C)) = P(A)P(B \cup C)$
2.  $P(AB) \neq P(A)P(B)$
3.  $P(AC) \neq P(A)P(C)$

**构造如下：**

设样本空间为掷一枚均匀骰子的结果，即 $\Omega = \{1, 2, 3, 4, 5, 6\}$，其中每个基本事件发生的概率均为 $1/6$。

定义事件如下：
*   $A = \{1, 2, 3\}$ （掷出点数小于 4）
*   $B = \{3\}$ （掷出点数为 3）
*   $C = \{4\}$ （掷出点数为 4）

**验证步骤：**

1.  **计算各事件概率：**
    $$P(A) = \frac{3}{6} = \frac{1}{2}$$
    $$P(B) = \frac{1}{6}$$
    $$P(C) = \frac{1}{6}$$

2.  **验证 A 与 B 的独立性：**
    积事件 $AB = A \cap B = \{3\}$。
    $$P(AB) = \frac{1}{6}$$
    $$P(A)P(B) = \frac{1}{2} \times \frac{1}{6} = \frac{1}{12}$$
    因为 $P(AB) \neq P(A)P(B)$，所以 **A 与 B 不独立**。

3.  **验证 A 与 C 的独立性：**
    积事件 $AC = A \cap C = \emptyset$（空集）。
    $$P(AC) = 0$$
    $$P(A)P(C) = \frac{1}{2} \times \frac{1}{6} = \frac{1}{12}$$
    因为 $P(AC) \neq P(A)P(C)$，所以 **A 与 C 不独立**。

4.  **验证 A 与 (B $\cup$ C) 的独立性：**
    和事件 $B \cup C = \{3, 4\}$。
    $$P(B \cup C) = \frac{2}{6} = \frac{1}{3}$$
    计算 $A$ 与和事件的积事件：
    $$A \cap (B \cup C) = \{1, 2, 3\} \cap \{3, 4\} = \{3\}$$
    $$P(A \cap (B \cup C)) = \frac{1}{6}$$
    检查独立性条件：
    $$P(A) \times P(B \cup C) = \frac{1}{2} \times \frac{1}{3} = \frac{1}{6}$$
    因为 $P(A \cap (B \cup C)) = P(A)P(B \cup C)$，所以 **A 与 (B $\cup$ C) 独立**。

**结论：** 该例子满足题目所有要求。

---

#### （2）设 A，B 两个事件的概率均大于 3/4。证明 A 与 B 的积事件的概率大于 1/2。

**证明：**

根据概率的加法公式，对于任意两个事件 $A$ 和 $B$，有：
$$P(A \cup B) = P(A) + P(B) - P(AB)$$

移项可得积事件概率的表达式：
$$P(AB) = P(A) + P(B) - P(A \cup B)$$

已知条件为：
$$P(A) > \frac{3}{4}, \quad P(B) > \frac{3}{4}$$

根据概率的公理化定义，任何事件的概率不超过 1，即：
$$P(A \cup B) \le 1$$

将上述不等式代入积事件的表达式中，可得：
$$
\begin{aligned}
P(AB) &= P(A) + P(B) - P(A \cup B) \\
&\ge P(A) + P(B) - 1
\end{aligned}
$$

将已知条件的数值代入：
$$
\begin{aligned}
P(AB) &> \frac{3}{4} + \frac{3}{4} - 1 \\
&= \frac{6}{4} - 1 \\
&= \frac{3}{2} - 1 \\
&= \frac{1}{2}
\end{aligned}
$$

即：
$$P(AB) > \frac{1}{2}$$

**证毕。**

### 答案

1.  **反例**：取 $\Omega=\{1,2,3,4,5,6\}$ 为等概样本空间，$A=\{1,2,3\}$，$B=\{3\}$，$C=\{4\}$。此时 $A$ 与 $B \cup C$ 独立，但 $A$ 与 $B$、$A$ 与 $C$ 均不独立。
2.  **证明**：利用 $P(AB) = P(A)+P(B)-P(A \cup B) \ge P(A)+P(B)-1$，代入 $P(A)>3/4, P(B)>3/4$ 即可得证 $P(AB) > 1/2$。

### 知识延伸

1.  **独立性的本质**：
    *   事件的独立性是纯粹的概率属性，与事件在集合论上的关系（如包含、互斥）不同。
    *   $A$ 与 $B \cup C$ 独立并不意味着 $A$ 与 $B$ 或 $A$ 与 $C$ 独立；反之，即使 $A$ 与 $B$ 独立且 $A$ 与 $C$ 独立，也不一定能推出 $A$ 与 $B \cup C$ 独立（除非 $B$ 和 $C$ 互斥，或者满足更强的相互独立条件）。

2.  **邦费罗尼不等式 (Bonferroni Inequalities)**：
    *   本题第二问是邦费罗尼不等式的一阶形式。
    *   通用公式：$P(\bigcap_{i=1}^n A_i) \ge \sum_{i=1}^n P(A_i) - (n-1)$。
    *   该不等式常用于在已知各个事件边缘概率的情况下，估计它们同时发生概率的下界。

3.  **易错点**：
    *   混淆“互斥”（Disjoint）与“独立”（Independent）。互斥是指 $A \cap B = \emptyset$，若 $P(A)>0, P(B)>0$，互斥一定导致不独立（因为 $P(AB)=0 \neq P(A)P(B)$）。
    *   在证明不等式时，忘记 $P(A \cup B) \le 1$ 这一隐含的上界条件。
