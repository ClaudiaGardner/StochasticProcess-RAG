# 习题4.8

## 题目

假设 $\{N ( t ) ; t > 0 \}$ 是一个更新计数过程，间隔时间随机变量序列为 $\{X_{n} ; n \geqslant1 \}$ 。不论是否 $\overline{{X}} < \infty$ ， 我们有 $\operatorname* {l i m}_{t \to\infty} N ( t )=\infty\mathrm{\ \ W P 1}$ 和 $\operatorname* {l i m}_{t \to\infty} \mathbf{E} [ N ( t ) ]=\infty$ . (1)请论证每个样本函数都有极限，或有限，或无穷。(因为样本函数是不减的） (2)请论证 $\{N ( t ) < n \}$ 这个事件应包含事件 $\left\{\operatorname* {l i m}_{t \to\infty} N ( t ) < n \right\}$ 所以， $\operatorname* {P r} \{\operatorname* {l i m} N ( t ) < n \} \leq P \{N ( t ) < n \}=$ $P \{S_{n} > t \} \to0$ ，因为 $S_{n}$ 是随机变量。 (3)证明 $\operatorname* {l i m}_{t \to\infty} \mathbf{E} [ N ( t ) ]=\infty$ 。

### 题目分析

本题考查**更新过程（Renewal Process）**的基础渐近性质。主要涉及以下核心概念：
1.  **更新计数过程 $N(t)$ 的定义**：$N(t) = \sup\{n : S_n \le t\}$，其中 $S_n$ 是第 $n$ 次更新发生的时刻。
2.  **样本路径的单调性**：利用 $N(t)$ 关于时间 $t$ 的非减性质来确定极限的存在性。
3.  **事件包含关系与概率极限**：通过事件之间的逻辑关系，利用 $S_n$ 的分布性质证明 $N(t)$ 依概率 1 趋于无穷。
4.  **期望的极限**：利用单调收敛定理或概率不等式证明更新函数 $m(t) = \mathbf{E}[N(t)]$ 趋于无穷。

本题的目标是不依赖于平均间隔时间 $\mathbf{E}[X]$ 是否有限，证明更新次数及其期望随时间推移必然趋向于无穷。

### 解题过程

#### (1) 论证每个样本函数都有极限

对于更新计数过程 $\{N(t); t > 0\}$，其定义为在时间区间 $(0, t]$ 内发生的更新次数。
对于任意固定的样本路径 $\omega$，考察函数 $N(t, \omega)$ 关于 $t$ 的性质：
由于 $N(t)$ 计数的是累积发生的事件数，对于任意 $0 < t_1 < t_2$，时间区间 $(0, t_1]$ 是 $(0, t_2]$ 的子集。因此，在 $(0, t_2]$ 内发生的更新次数必然大于等于在 $(0, t_1]$ 内发生的次数。
即：
$$
N(t_1) \le N(t_2), \quad \forall t_1 < t_2
$$
这意味着 $N(t)$ 是关于 $t$ 的**单调不减**函数。
根据数学分析中的单调有界定理的推广，单调不减函数在 $t \to \infty$ 时，其极限必然存在，该极限要么是一个有限的数，要么是 $+\infty$。
即对于每个样本路径，$\lim_{t \to \infty} N(t)$ 存在且：
$$
\lim_{t \to \infty} N(t) \in \{0, 1, 2, \dots\} \cup \{+\infty\}
$$

#### (2) 论证 $\{N(t) < n\}$ 包含 $\{\lim_{t \to \infty} N(t) < n\}$ 并证明概率为 0

**步骤 1：建立事件包含关系**
定义事件 $A = \left\{ \omega : \lim_{t \to \infty} N(t, \omega) < n \right\}$。
由于 $N(t)$ 是取非负整数值的单调不减函数，如果其极限小于 $n$（即极限最大为 $n-1$），那么对于该路径上的任意时刻 $t > 0$，函数值 $N(t)$ 都不可能达到或超过 $n$。
换言之，若 $\lim_{t \to \infty} N(t) < n$，则必然有：
$$
N(t) < n, \quad \forall t > 0
$$
因此，对于任意固定的 $t$，事件 $A$ 必然包含在事件 $\{N(t) < n\}$ 中：
$$
\left\{ \lim_{t \to \infty} N(t) < n \right\} \subseteq \{ N(t) < n \}
$$

**步骤 2：利用概率不等式与 $S_n$ 的性质**
根据概率的单调性，子事件的概率小于等于父事件的概率：
$$
\operatorname{Pr}\left\{ \lim_{t \to \infty} N(t) < n \right\} \le P\{ N(t) < n \}
$$
根据更新过程的定义，事件 $\{N(t) < n\}$ 等价于第 $n$ 次更新发生的时刻 $S_n$ 大于 $t$。即如果在 $t$ 时刻更新次数还不到 $n$，说明第 $n$ 次更新还在 $t$ 之后：
$$
\{ N(t) < n \} \iff \{ S_n > t \}
$$
因此：
$$
\operatorname{Pr}\left\{ \lim_{t \to \infty} N(t) < n \right\} \le P\{ S_n > t \}
$$
上式对任意 $t > 0$ 均成立。现在令 $t \to \infty$。
由于 $S_n = \sum_{i=1}^n X_i$ 是 $n$ 个非负随机变量的和（假设 $X_i$ 不全为0），$S_n$ 是一个取值有限的随机变量（即 $P(S_n < \infty) = 1$）。对于任意取值有限的随机变量，其分布函数的尾部概率随 $t$ 趋于无穷而趋于 0：
$$
\lim_{t \to \infty} P\{ S_n > t \} = 0
$$
由此可得：
$$
\operatorname{Pr}\left\{ \lim_{t \to \infty} N(t) < n \right\} \le 0
$$
因为概率非负，所以：
$$
\operatorname{Pr}\left\{ \lim_{t \to \infty} N(t) < n \right\} = 0
$$
这意味着 $N(t)$ 的极限是一个有限数的概率为 0。由于极限必然存在（由第一问可知），故：
$$
\operatorname{Pr}\left\{ \lim_{t \to \infty} N(t) = \infty \right\} = 1 \quad (\text{WP1})
$$

#### (3) 证明 $\lim_{t \to \infty} \mathbf{E}[N(t)] = \infty$

我们需要证明更新函数 $m(t) = \mathbf{E}[N(t)]$ 发散。

**方法一：利用单调收敛定理**
由 (1) 知 $N(t)$ 关于 $t$ 单调不减，且 $N(t) \ge 0$。
由 (2) 知 $N(t) \to \infty$ almost surely。
根据单调收敛定理（Monotone Convergence Theorem），对于非负单调增的随机变量序列，期望的极限等于极限的期望：
$$
\lim_{t \to \infty} \mathbf{E}[N(t)] = \mathbf{E}\left[ \lim_{t \to \infty} N(t) \right] = \mathbf{E}[\infty] = \infty
$$

**方法二：利用定义直接推导（不依赖测度论定理）**
对于任意给定的正整数 $M$，考虑事件 $\{N(t) \ge M\}$。
$$
\mathbf{E}[N(t)] = \sum_{k=1}^{\infty} P(N(t) \ge k)
$$
由于各项均为非负，我们可以截断求和：
$$
\mathbf{E}[N(t)] \ge \sum_{k=1}^{M} P(N(t) \ge k)
$$
由 (2) 的结论可知，对于任意固定的 $k$，$\lim_{t \to \infty} P(N(t) < k) = 0$，这意味着 $\lim_{t \to \infty} P(N(t) \ge k) = 1$。
对不等式两边取下极限（$\liminf_{t \to \infty}$）：
$$
\liminf_{t \to \infty} \mathbf{E}[N(t)] \ge \sum_{k=1}^{M} \lim_{t \to \infty} P(N(t) \ge k) = \sum_{k=1}^{M} 1 = M
$$
由于 $M$ 是任意选取的正整数，令 $M \to \infty$，可得：
$$
\lim_{t \to \infty} \mathbf{E}[N(t)] = \infty
$$

### 答案

(1) 因为 $N(t)$ 是计数过程，对于任意 $t_1 < t_2$，有 $N(t_1) \le N(t_2)$。单调函数必有极限（有限或 $+\infty$）。

(2) 因为 $N(t)$ 单调不减且取整数值，若极限小于 $n$，则所有 $t$ 时刻的值都必须小于 $n$。因此 $\left\{\lim_{t \to \infty} N(t) < n \right\} \subseteq \{N(t) < n\}$。
推导如下：
$$
\operatorname{Pr}\left\{ \lim_{t \to \infty} N(t) < n \right\} \le \lim_{t \to \infty} P\{N(t) < n\} = \lim_{t \to \infty} P\{S_n > t\} = 0
$$
故 $\operatorname{Pr}\{\lim_{t \to \infty} N(t) = \infty\} = 1$。

(3) 利用单调收敛定理或期望的定义：
$$
\lim_{t \to \infty} \mathbf{E}[N(t)] = \mathbf{E}\left[\lim_{t \to \infty} N(t)\right] = \infty
$$

### 知识延伸

1.  **更新过程的基本性质**：本题证明了更新过程最基本的性质——只要间隔时间不是恒为0，更新次数随时间无限增加，且平均更新次数也无限增加。这是后续讨论更新定理的基础。
2.  **重要公式**：
    *   关系式：$N(t) < n \iff S_n > t$。这是连接计数过程 $N(t)$ 和部分和序列 $S_n$ 的桥梁。
    *   更新函数：$m(t) = \mathbf{E}[N(t)] = \sum_{n=1}^\infty P(S_n \le t)$。
3.  **相关定理**：
    *   **更新过程强大数定律 (SLLN for Renewal Processes)**：$\frac{N(t)}{t} \to \frac{1}{\mu}$ a.s.，其中 $\mu = \mathbf{E}[X_1]$。本题结论是该定理的前提（即 $N(t) \to \infty$）。
    *   **基本更新定理 (Elementary Renewal Theorem)**：$\lim_{t \to \infty} \frac{\mathbf{E}[N(t)]}{t} = \frac{1}{\mu}$。
4.  **易错点**：混淆 $N(t) \to \infty$ (a.s.) 和 $\mathbf{E}[N(t)] \to \infty$。虽然前者通常蕴含后者（在单调情况下），但严格证明需要依据单调收敛定理或截断求和的方法。
