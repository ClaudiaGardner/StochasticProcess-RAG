### 题目分析

本题考查随机过程中**计数过程（Counting Process）** $N(t)$ 与**到达时刻（Arrival Times）** $S_n$ 之间的基本逻辑关系。

1.  **符号定义**：
    *   $N(t)$：表示在时间区间 $(0, t]$ 内发生的事件总数。
    *   $S_n$：表示第 $n$ 个事件发生的时刻（即前 $n$ 个这一事件间隔之和）。通常定义 $S_0 = 0$。
    *   题目中关于泊松分布的公式 $P\{X_n=k\} = e^{-\mu}\frac{\mu^k}{k!}$ 似乎是对计数过程分布特性的补充说明（即 $N(t)$ 服从泊松分布），或者是题目背景信息的残留，但这不影响 $N(t)$ 与 $S_n$ 定义上的逻辑等价性判断。

2.  **核心考点**：
    *   理解事件 $\{N(t) \geq n\}$ 与事件 $\{S_n \leq t\}$ 的等价关系。
    *   利用集合论中的补集和不等式性质判断命题真伪。

3.  **求解目标**：
    *   判断 (1), (2), (3) 三个逻辑命题是否正确。

### 解题过程

在随机过程中，计数过程 $N(t)$ 与到达时刻 $S_n$ 满足以下基本定义关系：
$$N(t) = \sup \{ n \geq 0 : S_n \leq t \}$$
这意味着，$N(t)$ 是在时刻 $t$ 之前（含 $t$）发生的最大的事件编号。由此可得出一个最基础的等价关系（**基本引理**）：
$$N(t) \geq n \iff S_n \leq t$$
**解释**：“在时刻 $t$ 截止时，事件发生的数量至少为 $n$ 个” 等价于 “第 $n$ 个事件发生的时刻 $S_n$ 早于或等于 $t$”。

基于此基本引理，我们逐一分析三个命题：

#### (1) 分析命题：$N(t) < n \iff S_n > t$

我们从基本引理出发，取其对立事件（补集）：
1.  已知基本关系：$\{ N(t) \geq n \} \iff \{ S_n \leq t \}$。
2.  对等式两边同时取补集：
    $$ \{ N(t) \geq n \}^c \iff \{ S_n \leq t \}^c $$
3.  不等式的补集分别为：
    *   $N(t) \geq n$ 的补集是 $N(t) < n$。
    *   $S_n \leq t$ 的补集是 $S_n > t$。
4.  因此得出：
    $$ N(t) < n \iff S_n > t $$

**结论**：该命题是**正确**的。

#### (2) 分析命题：$N(t) \leq n \iff S_n \geq t$

我们需要验证左右两边是否互为充要条件。
1.  **左边**：$N(t) \leq n$。
    这等价于 $N(t) < n+1$。根据命题(1)的推导，这等价于 $S_{n+1} > t$。即：第 $n+1$ 个事件发生在时刻 $t$ 之后。
2.  **右边**：$S_n \geq t$。
    这表示第 $n$ 个事件发生在时刻 $t$ 之后（或恰好在 $t$）。
3.  **比较**：
    *   如果 $N(t) = n$，则根据定义必须满足 $S_n \leq t$ 且 $S_{n+1} > t$。
    *   在这种情况下（$N(t)=n$），左边命题 $N(t) \leq n$ 成立。
    *   但是，右边命题要求 $S_n \geq t$。由于我们已知 $S_n \leq t$，除非 $S_n = t$，否则 $S_n \geq t$ 通常不成立（特别是当 $S_n < t$ 时）。
    *   **反例**：假设 $n=5$。如果在时刻 $t$ 发生了 5 个事件（$N(t)=5$），且第 5 个事件发生在 $t/2$ 时刻（即 $S_5 = t/2 < t$）。此时 $N(t) \leq 5$ 成立，但 $S_5 \geq t$ 不成立。

**结论**：该命题是**错误**的。

#### (3) 分析命题：$N(t) > n \iff S_n < t$

1.  **左边**：$N(t) > n$。
    由于 $N(t)$ 是整数，这等价于 $N(t) \geq n+1$。
    根据基本引理，这等价于 $S_{n+1} \leq t$。
2.  **右边**：$S_n < t$。
3.  **比较**：
    *   我们有推导链：$N(t) > n \iff S_{n+1} \leq t$。
    *   由于 $S_n < S_{n+1}$（假设事件间隔非负且不为0），若 $S_{n+1} \leq t$，则必然有 $S_n < t$。所以 左 $\implies$ 右 成立。
    *   但是，反之是否成立？若 $S_n < t$，是否意味着 $N(t) > n$？
    *   **反例**：假设 $S_n < t < S_{n+1}$。此时第 $n$ 个事件发生在 $t$ 之前，但第 $n+1$ 个事件发生在 $t$ 之后。根据定义，$N(t) = n$。
    *   在这种情况下，$S_n < t$ 成立（右边为真），但 $N(t) = n$ 并不满足 $N(t) > n$（左边为假）。

**结论**：该命题是**错误**的。

### 答案

(1) **正确**。
$$N(t) < n \iff S_n > t$$

(2) **不正确**。
正确的关系应为：$N(t) \leq n \iff S_{n+1} > t$。

(3) **不正确**。
正确的关系应为：$N(t) > n \iff S_{n+1} \leq t$。

### 知识延伸

1.  **基本对偶关系**：
    在处理泊松过程或更新过程时，最核心的公式是：
    $$ \{ N(t) \geq n \} \iff \{ S_n \leq t \} $$
    所有的变体都应基于此公式及其补集进行推导。

2.  **直观理解**：
    *   $N(t)$ 是画在纵轴上的阶梯函数，$S_n$ 是阶梯跳跃发生的横轴坐标。
    *   $N(t) < n$ 意味着“还没到第 $n$ 个跳跃”，所以第 $n$ 个跳跃的时间 $S_n$ 必须在当前时间 $t$ 的右边（$S_n > t$）。

3.  **易错点**：
    *   混淆严格不等号 $<$ 与非严格不等号 $\leq$。在连续时间分布（如泊松过程）中，$P(S_n = t) = 0$，不等号是否包含等号通常不影响概率值，但在逻辑等价性判断中必须严格区分。
    *   混淆下标 $n$ 与 $n+1$。例如 $N(t) \leq n$ 实际上是在描述第 $n+1$ 个事件的状态，而不是第 $n$ 个。
